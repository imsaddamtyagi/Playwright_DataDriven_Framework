# Playwright Data-Driven Automation Framework

## Overview
This is a Data-Driven Automation Framework built on Playwright with JavaScript (Node.js) and on the VS Code IDE. It supports data-driven testing and aims to provide a robust solution for automated testing.

## Features
- **Data-Driven Testing**: Easily manage test data using external data sources.
- **HTML Reports**: Generate comprehensive HTML reports for test results.
- **Exception Handling**: Gracefully handle exceptions during test execution.
- **Version Control**: Seamless integration with GitHub for version control.

## Installation
To get started with this framework, follow these steps:
1. Clone the repository: `git clone https://github.com/imsaddamtyagi/Playwright_DataDriven_Framework.git`
2. Install dependencies: `npm install`
3. Run tests: `npx playwright test`

## Usage
- Enter test data in the provided Excel file.
- Add XPath and test data to the Excel file.
- Write your test scripts in the `tests` folder.
- Configure your project settings in `playwright.config.js`.
